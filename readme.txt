===Description===
Плагин для лёгкого подключения push-уведомлений на вашем сайте. Пройдя двухминутную регистрацию прямо через Wordpress, вы сможете включить подписку на push на вашем сайте одной кнопкой. Плагин поддерживает два способа отправки пуш: автоматический - при создании или обновлении записи на блоге, и ручную рассылку по подписчикам.
